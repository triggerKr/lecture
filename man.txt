산출물 요구사항
※ RFP에 명시 되지 않더라도 프로젝트 진행을 위해 필 요한 사항은 추가 요청 될 수 있음.
※ RFP에 명시 되지 않은 품질 요구사항에 대해서 프로 젝트 초기 워크샵을 통해 추가 도출 될 수 있으며, 해당 항목에 대한 개발 내용은 포함되어야 함
※ 작성된 코드는 DS 코드 품질 지수를 만족해야 함
•   정적분석 Major/Minor 결함 0건
•   SAM 지수 4.0 이상
(평균이 아닌 각 항목별 4.0 이상)
•   TEM 지수 4.0 이상
(TEM은 code coverage로 산출되며 대략적으로 line coverage * function coverage * test 성공율 로 계산됨)
